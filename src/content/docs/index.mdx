## Features

| Feature | Description |
|---------|-------------|
| **Admin Panel** | Interactive UI to manage players in real-time |
| **Moderation** | Kick, ban, mute, and player monitoring |
| **Teleportation** | Advanced teleportation between players and locations |
| **Docker Images** | Multi-arch images with automatic Hytale version updates |
| **Helm Charts** | Production-ready Kubernetes deployment |
